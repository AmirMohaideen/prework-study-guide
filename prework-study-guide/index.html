HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The li, ol, and ul commands are used for bullet points/unordered lists
  • The h1,h2,h3,... command is used for making headers
  • The p command is used for making paragraphs
  • The img command is used for adding an image to the page along with all the information it is using
  • The br command creates a line of blank space used to seperate blocks of content
  • The a command creates and anchor for a hyperlink, the href attribute points to the URL for the link

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card: the period makes card a class which will grant what ever has it as a class gains the attributes of card.

Comand line

  • cd (directory name): change directory
  • cd ..: goes back a directory
  • cp (file) (directory): copies the file and puts it into the directory
  • ls: lists all the files and directories in the working directory
  • mkdir (name): Make directory witht he name you chose
  • mv (file) (directory): moves the file into the directory (I tried this and I dont know how it works)
  • pwd: print working directory, tells you what directory you are in
  • rm (file): deletes named file
  • rm -r: deletes a directory and directories that come from it
  • touch (file name): creates a file inside the working directory with the name you chose

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone (link) clones a file into the current repository
  • git add -A: any changes we made are saved? (I dont really know what this does)
  • git commit -m "comment of the changes you made/reason for the commit": get file ready for push? adds a comment of reason for this push
  • git push origin main: commits the changes you made and puts them on the main branch
